Logo Concepts

Three directions for an animated GH mark. Each uses SVG stroke animation — the same file works for web (animated) and print (static). Click Replay to watch the draw again.

01
GH — Pure Monogram

The initials as a single geometric mark. Architectural, minimal. G draws first, H follows. Gold on dark for screen; black on white for print. The most versatile — works at any size from a favicon to a banner.

Print / Light

Design notes

  • G draws in gold — it's the primary mark. H follows in white, subordinate.
  • Stroke-width 2.2px at native size. Scale freely — SVG is resolution-independent.
  • Animation: G 1.4s, H stems staggered, total runtime ~2.0s.
  • For print: export as PDF or EPS. The SVG static state is the finished logo.
02
GH + Wordmark

The monogram mark with "Greg Hobden" set below in Outfit light. A rule line draws between them. Ideal for letterheads, email signatures, and the site footer — contexts where the full name adds clarity.

Web / Screen
Print / Light

Design notes

  • Rule line draws after the mark completes — a deliberate pause before the name appears.
  • "GREG HOBDEN" fades in rather than drawing — text-to-path is complex; opacity fade is cleaner.
  • "FILM PRODUCER" in gold provides the subtitle / discipline line.
  • On paper: perfect for letterhead — mark upper-left, wordmark centred or aligned.
03
GH — Framed Mark

The initials sit inside a rectangular frame that draws itself first — like a title card coming to life. More structured and cinematic. References the format of film slates and title sequences. Strong as a stamp or seal on print materials.

Web / Screen
Print / Light

Design notes

  • Frame draws first (1.2s) — the container arrives before the content. Deliberately cinematic.
  • Corner accent marks reference film/camera viewfinder aesthetics.
  • GH draws inside the frame as a second act. Total runtime ~2.4s.
  • Works beautifully as a wax seal, embossed stamp, or business card centrepiece on print.
  • On screen: could sit in the hero section as a pre-loader mark before content reveals.
