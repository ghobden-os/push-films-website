Name layout: "Greg" and "Hobden" sit on separate lines with very tight leading (line height 0.85 — the lines almost touch). This is intentional.

"Hobden" italic: Select just the word Hobden and apply italic. Do not apply italic to "Greg".

Disciplines layout: Each discipline sits on its own line. The yellow dot ( · ) is on its own line between each, centred. Stack them vertically.

Colour format: Hex values above. In Adobe tools: #F5D800 = R 245 / G 216 / B 0. #EDEDE8 = R 237 / G 237 / B 232. #BBBBBB = R 187 / G 187 / B 187.
