How to install — Apple Mail

  • Open Mail → Settings → Signatures
  • Click + to create a new signature
  • Open the light-background preview above in your browser, select the entire signature (click and drag), and copy it (⌘C)
  • Paste directly into the signature box in Mail — the formatting carries over
  • Assign the signature to your account in the left column

How to install — Gmail

  • In Gmail: Settings (gear icon) → See all settings → General → scroll to Signature
  • Click Create new
  • In the signature editor, click the HTML source icon (looks like <>) if available — or use the copy-paste method below
  • Select and copy the signature from the light preview above, paste into the Gmail signature box
  • Note: Gmail strips some CSS. The disc may render as a square in older Gmail clients (the yellow colour and G still show). The overall layout will hold.

How to install — Outlook (Mac)

  • Outlook → Preferences → Signatures → +
  • Select and copy from the light preview above, paste into the editor
  • Outlook on Windows uses Word rendering — border-radius is not supported, so the disc will appear as a square. Still fully branded; just not circular.

Compatibility notes

  • The yellow disc is CSS-rendered — it's circular in Apple Mail, Gmail (web), and modern Outlook (Mac). Outlook on Windows renders it square.
  • DM Sans loads from Google Fonts in browsers; email clients fall back to system fonts (-apple-system / Segoe UI) which look very similar.
  • The yellow dots between contact details are a brand touch — if they appear as boxes in a client, replace them with a plain | separator.
  • Links are unstyled (text-decoration:none) — some clients will override this and show underlines. That's fine.
Tip: To use the dark-background version, your email itself would need a dark background — unusual in standard email. Use the light version by default. The dark version is here for reference or for future branded email templates.
